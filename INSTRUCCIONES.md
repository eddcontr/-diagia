# DIAGIA v2.1 — Instrucciones de Instalación

## Paso 1 — Instalar Node.js (si no lo tienes)
1. Ve a https://nodejs.org
2. Descarga la versión LTS (recomendada)
3. Instálala normalmente

## Paso 2 — Subir a Vercel (más fácil)

### Opción A — Sin código (recomendada)
1. Ve a https://vercel.com y crea cuenta gratis con tu email
2. Ve a https://github.com y crea cuenta gratis
3. Crea un repositorio nuevo llamado "diagia"
4. Sube todos estos archivos al repositorio
5. En Vercel conecta tu GitHub y selecciona el repositorio "diagia"
6. Vercel lo despliega automático y te da una URL como: diagia.vercel.app

### Opción B — Desde tu computadora
1. Abre terminal/cmd en la carpeta del proyecto
2. Ejecuta: npm install
3. Ejecuta: npm install -g vercel
4. Ejecuta: vercel
5. Sigue las instrucciones — te da la URL al final

## Paso 3 — Abrir desde cualquier dispositivo
- Abre la URL en tu Dell 5410 en el taller
- Abre la URL en tu Samsung S24 Ultra
- En Chrome Android: menú → "Agregar a pantalla de inicio" → funciona como app

## Notas importantes
- La app necesita internet para funcionar (usa la API de Claude)
- Cada sesión de diagnóstico es independiente
- Usa "NUEVO CASO" para cada vehículo diferente
- Puedes subir fotos del escáner directamente desde el celular
